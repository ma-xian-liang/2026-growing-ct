# The Breathing Cosmos: Growing Speed of Light Theory
## Project Overview
This open-source theoretical physics project establishes a novel cosmic evolution framework centered on **time-varying speed of light \(c(t)\)**. The core equation \(G(t)c(t)^2 = P\) reinterprets gravitational constant \(G\) as a time-dependent derived quantity, while keeping the cosmic coupling constant \(P\) and fine-structure constant \(\alpha\) permanently invariant.

This model revises the standard \(\Lambda\)CDM cosmology, re-examines fundamental physical constants, resolves multiple long-standing cosmological puzzles, and provides a new path toward unified field theory.

## Core Theoretical Postulates
1. The speed of light \(c\) is a time-dependent variable \(c(t)\); the cosmic invariant coupling constant \(P\) satisfies the core relation:
\[
G(t)\,c(t)^2 = P
\]
Gravitational constant \(G(t)\) is no longer a fundamental constant, but a quantity evolving synchronously with light speed.
2. The fine-structure constant \(\alpha\) remains unchanged throughout all cosmic time.

### Scaling Law of \(c(t)\)
\(c(t)\) scales proportionally with the cosmic scale factor \(a(t)\) and redshift factor \(1+z\) in standard cosmology:
\[
c(t) \propto a(t) \propto (1+z)
\]

## Critical Cosmological Predictions
### 1. CMB Decoupling Epoch (\(1+z=1100\))
- Early light speed: \(c_{\text{early}} = c_0 / 1100\) (\(c_0\) = present-day light speed)

### 2. Cosmic Timescale
The model extends the cosmic formation timeline to **[-59.1, -45.2] Gyr**, far longer than the standard model’s 13.8 Gyr universe age. It supports a slow, gently evolving universe instead of a violent singular Big Bang.

### 3. Early Spacetime Curvature
Revised Einstein curvature tensor \(G_{\mu\nu}\) analysis reveals extreme spacetime curvature in two key eras:
1. Early galaxy formation epoch (\(1+z=25\))
2. CMB decoupling epoch (\(1+z=1100\))

### 4. Cosmic Coupling Constant \(P\)
Fundamental relation linking spacetime curvature, mass-energy distribution and variable light speed:
\[
P = \frac{c(t)^6 G_{\mu\nu}}{8\pi T_{\mu\nu}} \approx \frac{c(t)^4 G_{\mu\nu}}{8\pi M_{\mu\nu}}
\]
\(P\) describes the intrinsic coupling between cosmic mass-energy, spacetime geometry, and time-varying light speed. Macroscopically, matter distribution dynamically controls both spacetime curvature and light propagation velocity.

### 5. Present-Day Light Speed Variation
Annual changing rate of \(c(t)\) measured at current cosmic time:
\[
\Delta c / \text{yr} = [9.43,\ 20.7]\ \mathrm{mm\cdot s^{-1}\cdot yr^{-1}}
\]
The paper proposes the High Energy Photon Source (HEPS) as an experimental facility to verify this tiny light speed drift.
